# v0.9.0 Persona Adoption Scorecard and Feature Priority Matrix

_Last reviewed: 2026-03-05 (baseline commit ad788a4)._

This scorecard translates product direction into measurable adoption expectations by persona and maps v0.9.0 feature priorities to those personas.

## 1) Persona Adoption Scorecard

| Persona | Current Adoption Fit (0-10) | v0.9.0 Target Fit (0-10) | Main Value Driver | Current Friction | Required v0.9.0 Tickets |
| --- | --- | --- | --- | --- | --- |
| Privacy-native technical users | 8.0 | 8.8 | E2EE + decentralized + local-first control | Community structure and call reliability gaps | `COMM-001`, `COMM-002`, `SEARCH-001`, `CALL-001`, `CALL-002`, `CALL-003` |
| Everyday private chat users | 5.5 | 7.0 | Simple private chat with quality media and calls | Onboarding friction, no stable calls, weak retrieval | `UX-001`, `DM-001`, `SEARCH-001`, `CALL-002`, `CALL-004` |
| Work pods and small teams | 6.5 | 8.0 | Private collaboration without central admin lock-in | Missing channels/threads, weak knowledge layer | `COMM-001`, `COMM-002`, `KNOW-001`, `FILES-001`, `COORD-001`, `COORD-003` |
| Encrypted niche community admins | 8.0 | 9.0 | Invite-only governance with privacy-first operations | Governance ergonomics and onboarding scale limits | `GOV-001`, `MEM-001`, `UX-001`, `KNOW-001`, `PERF-001` |
| Content-sharing communities (music/literature/video/comics) | 7.0 | 8.5 | Rich media sharing in private community contexts | Media discovery, curation, and file organization limits | `MEDIA-001`, `FILES-001`, `SAVE-001`, `SEARCH-001`, `COMM-001` |

## 3) v0.9.0 Success Metrics by Persona

| Persona | Metric | Target |
| --- | --- | --- |
| Everyday private chat users | D7 retained conversations per active user | +20% vs v0.8.x baseline |
| Work pods and small teams | Weekly active teams with 3+ members and 2+ spaces used | +30% vs v0.8.x baseline |
| Encrypted niche community admins | Communities with configured governance mode and onboarding checklist | 60% of active communities |
| Content-sharing communities | Sessions using media albums or file cabinet | 35% of active community sessions |
| All personas | Message timeline p95 UI update latency under load | < 100ms |
| All call users | 1:1 call setup success rate | >= 90% across supported networks |
