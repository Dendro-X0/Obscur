## T1/T2 — Locked filesystem (desktop, custom data root)

Prereq: data root on non-default path (e.g. `E:\app.obscur.desktop` via junction).

1. Unlock Obscur; send a DM and cache one vault thumb.
2. **Lock** (title bar or auto-lock).
3. In Explorer, open the data folder, or run:
   ```bash
   node scripts/audit-data-root-at-rest.mjs "E:/app.obscur.desktop"
   ```
   Expect **PASS** while locked.
4. Unlock — DM history and vault thumb still load.

Record: date, profile, paths inspected, pass/fail.

## Portable soak (two-device or same machine)

1. Quit Obscur completely.
2. Copy entire data root to USB or second folder.
3. On target machine: install Obscur → **Reconnect folder** → pick copied root.
4. Restart; unlock each profile; verify DM + groups + vault.
