# 32 v1.1.3 CP3 Voice Suite Replay Matrix

_Last reviewed: 2026-03-24 (baseline commit 796c992)._

This matrix defines the CP3 runtime evidence required for `v1.1.3` in the `M9` secure-voice lane.

Primary objective:
1. capture one-copy, reproducible evidence for both weak-network and account-switch voice session continuity,
2. verify async voice-note and delete-convergence diagnostics in the same evidence window,
3. reduce manual timing fragility before `v1.2.0` closeout.

## Canonical Helper

Preferred one-copy export:
: `copy(window.obscurM6VoiceReplay?.runCp3ReplaySuiteCaptureJson?.({ clearAppEvents: true, captureWindowSize: 400 }))`

Raw object probe:
: `window.obscurM6VoiceReplay?.runCp3ReplaySuiteCapture?.({ clearAppEvents: true, captureWindowSize: 400 })`

## Expected Bundle Shape

1. `weakNetwork`:
: replay + capture + `cp2EvidenceGate`.
2. `accountSwitch`:
: replay + capture + `cp2EvidenceGate`.
3. `suiteGate`:
: aggregate verdict over both paths.

Required fields to verify:
1. `weakNetwork.cp2EvidenceGate.pass === true`
2. `accountSwitch.cp2EvidenceGate.pass === true`
3. `suiteGate.pass === true`
4. `weakNetwork.replay.replayReadiness.readyForCp2Evidence === true`
5. `accountSwitch.replay.replayReadiness.readyForCp2Evidence === true`
6. `suiteGate.checks.weakAsyncVoiceSummaryPresent === true`
7. `suiteGate.checks.accountAsyncVoiceSummaryPresent === true`
8. `suiteGate.checks.weakDeleteSummaryPresent === true`
9. `suiteGate.checks.accountDeleteSummaryPresent === true`
10. `suiteGate.checks.weakAsyncVoiceStartFailureCountZero === true`
11. `suiteGate.checks.accountAsyncVoiceStartFailureCountZero === true`
12. `suiteGate.checks.weakDeleteRemoteFailureCountZero === true`
13. `suiteGate.checks.accountDeleteRemoteFailureCountZero === true`

## Supplemental Probes

1. Voice session summary:
: `window.obscurAppEvents?.getCrossDeviceSyncDigest?.(400)?.summary?.realtimeVoiceSession`
2. Async voice-note summary:
: `window.obscurAppEvents?.getCrossDeviceSyncDigest?.(400)?.summary?.asyncVoiceNote`
3. Delete convergence summary:
: `window.obscurAppEvents?.getCrossDeviceSyncDigest?.(400)?.summary?.deleteConvergence`
4. Transition slice:
: `window.obscurAppEvents?.getCrossDeviceSyncDigest?.(400)?.events?.["messaging.realtime_voice.session_transition"]`
5. Delete convergence slice:
: `window.obscurAppEvents?.getCrossDeviceSyncDigest?.(400)?.events?.["messaging.delete_for_everyone_remote_result"]`

## Pass Criteria

1. Suite helper returns both scenario bundles with no missing gate fields.
2. `suiteGate.failedChecks` is an empty array.
3. No `recovery_exhausted` signals in expected recoverable weak-network replay.
4. Account-switch path reports multi-room continuity evidence (room-switch counters present in replay).
5. No high-risk delete or voice-note failure signal in nominal run (`remoteFailedCount === 0`, `recordingStartFailedCount === 0`).

## Escalation Criteria

Escalate as blocker if either occurs:
1. `suiteGate.pass === false`,
2. repeated `cp2EvidenceGate.pass === false` in either replay path,
3. digest shows `realtimeVoiceSession.recoveryExhaustedCount > 0` in recoverable replay,
4. digest shows `deleteConvergence.remoteFailedCount > 0` in nominal replay.

Attach in escalation:
1. suite one-copy capture JSON,
2. `copy(window.obscurM6VoiceCapture?.captureJson(400))`,
3. `copy(window.obscurM0Triage?.captureJson(300))`,
4. short timestamped replay timeline.
