| Band | Themes |

## Maintainer rule: CI vs version

| Change | Where it lands | Tags? |
|--------|----------------|-------|
| `docs:check`, `release.yml` retries, typecheck fixes | `main` as they are fixed | **No** — not a version milestone |
| D1 + D3 + Test 9 + automated tests | `main` under v1.8.9 scope | **Yes** — single `v1.8.9` when gate green |

Do **not** open `v1.8.10` only to fix a failed Obscur Full Release preflight; fix on `main` and move `v1.8.9` tag forward if already pushed.
