## Purpose

Prove **pool + presets** under maintainer desktop soak:

1. Preset apply → configured endpoints → DM send/receive  
2. Multi-endpoint pool → kill one endpoint → **degraded** badge or honest failure (not silent green)  
3. Hybrid preset path (public Nostr + private HTTP mesh) optional row  

**Not in scope for this soak:** C13 Tor L3 · community roster · ACC-02.

## Row A — Preset UX (single window)

| Step | Action | Pass criterion |
|------|--------|----------------|
| A1 | Settings → Transport & connectivity | Transport packs catalog visible |
| A2 | Apply **High Redundancy** (or Default Stable) with existing list | Confirm dialog if list non-empty |
| A3 | Inspect active mix + adapter chips | Active/partial badges; Nostr WS chips on endpoints |
| A4 | Send DM marker `P14-L3-preset-YYYYMMDD` | Delivered or honest failure with status copy |

---

## Row B — Dual-window DM + pool (primary L3)

| Step | Action | Pass criterion |
|------|--------|----------------|
| B1 | Both profiles: same public preset (≥2 enabled relays) | Settings show matching enabled count |
| B2 | T1 → T2 send marker `P14-L3-dm-YYYYMMDD` | T2 receives ≤45s |
| B3 | T2 reply | T1 receives ≤45s |
| B4 | Settings pool badge | Healthy or degraded — **not** offline lie while DM works |
| B5 | Disable or block **one** enabled relay (Settings toggle off, or stop docker relay if used) | Badge → **degraded** or equivalent; refresh status |
| B6 | T1 → T2 send after kill | Still delivers via remaining relay **or** fails closed with honest copy |

Record: relay URLs, badge text, timestamps, marker strings.

---

## Row C — Hybrid preset (optional)

| Step | Action | Pass criterion |
|------|--------|----------------|
| C1 | Start mesh gateway `:8788` if not running | Health OK |
| C2 | Apply **Hybrid public + HTTP mesh** preset | Confirm + endpoints include `wss://` + `http://127.0.0.1:8788` |
| C3 | Dual-window DM marker | Send/receive or documented fail-closed reason |

Skip Row C if gateway unavailable — does not block 1.9.14 EXIT if Row B passes.

## Failure triage

| Symptom | Likely owner | First check |
|---------|--------------|-------------|
| Badge green but no receive | Pool subscribe / DM arm | Conduit panel + advanced endpoint list |
| Kill one relay, still “healthy” | P14-P partial pool mapping | `map-mesh-snapshot-to-relay-activity` |
| Preset apply no confirm | Empty relay list only skips dialog | `transport-preset-apply-policy` |
| Tor pack applies when Tor down | Slice B gate | Tor not ready badge + disabled Apply |
