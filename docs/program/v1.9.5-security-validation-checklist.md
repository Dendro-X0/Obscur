## Honest security claims (for v2.0 limitations sheet)

**May claim after v1.9.5 Pass:**

- E2E encrypted DM and sealed group messages on configured infrastructure.
- Recipient-local fraud **signals** (not judgments) on cold contacts.
- No Obscur-operated central message store.
- Workspace-scoped removal on operator-managed relays.

**Must not claim:**

- “Impossible to scam users” or “100% secure.”
- Global delete-for-everyone on open Nostr.
- Platform moderation or sender notification of warnings.
- Relay security for relays the user adds outside Obscur operator guidance.
