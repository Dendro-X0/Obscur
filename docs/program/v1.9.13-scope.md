## Problem statement

| Symptom | Impact |
|---------|--------|
| Chat→vault disabled since false-success reports | Users cannot save attachments from DM/group threads |
| G8 (L3/L4) unsigned | Cannot claim full “encrypted on disk” in public copy |

| L1 gate omits index/aggregator tests | Regressions in SQLite index path may slip through |

Secure Upload (Vault page) is the **demo-safe** path today. Core functionality for messaging users requires Phase 6 with proof.
