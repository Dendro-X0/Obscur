## Goal

Ship **v1.5.6** as:

1. **Mobile:** first **mobile product** milestone on the shared kernel — **M1** DM-first layout slice in the Android emulator / mobile shell (**not** a store or signing gate).
2. **Desktop / shared kernel:** reliability (R4, R5), invite UX (U1), themed selects, and **Phase 1 community** (descriptor truth, governance MVP, roster/display fixes) on the same version line while **M** is paused.

**Bar for success (full milestone):**

- **M1** — Auth → conversation list → thread navigable in mobile shell with shared hooks (no parallel DM path).
- Emulator smoke documented (M1–M2 subset); signing/F-Droid still optional.
- `release:test-pack` green before tag.

**Bar for success (desktop-only tag while M paused):** automated gates **A1–A4**, desktop regression **D1**, CHANGELOG accurate — see gate doc.
