```text
v1.8.14 (batch exit) ──► v1.9.0 (Lane K band) ──► v1.9.1 … v1.9.x ──► v2.0.0 (production demo)
```

### Tag checklist (maintainer)

1. Clean `main`: `pnpm -C apps/pwa typecheck` · `pnpm docs:check` · `pnpm release:test-pack -- --skip-preflight` · `pnpm version:check`
2. `pnpm version:bump minor` → workspace **1.9.0** (Tauri + Android versionCode **10900**)
3. Confirm [CHANGELOG.md](../../CHANGELOG.md) `## [1.9.0]` section matches [v1.9.0-release.md](../releases/v1.9.0-release.md)
4. Commit doc + version bump (maintainer request only)
5. `git tag v1.9.0` && `git push origin v1.9.0` (optional — GitHub Release not version truth)
6. Local installer: `pnpm desktop:package` → [local-desktop-packaging.md](./local-desktop-packaging.md)
7. Optional: `pnpm desktop:update-channel:publish` after signed build

**Do not** tag for CI-only fixes. **Do not** require Full Release workflow for underground distribution.
