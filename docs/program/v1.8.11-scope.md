## Safety defaults (B1)

- Bot may **publish only** — no expel, governance vote, or hide actions.
- Rate limit documented in runbook (e.g. max N messages/min per `communityId`).
- Bot pubkey must appear in descriptor allowlist before publish is accepted client-side.
- Honest copy: bot is visible as a member; not a hidden system account.
