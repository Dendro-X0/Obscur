## Goal

Ship **v1.5.5** as the first **mobile production** release on the **same version line** as desktop.

**Bar for success:**

- Signed Android APK on GitHub Release for tag `v1.5.5`.
- M1–M3 device matrix signed off.
- `versionName` and in-app version = `1.5.5`.
- Desktop regressions from v1.5.4 remain acceptable.
