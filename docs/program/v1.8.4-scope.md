## Goal

Close the post-REL-004 invite flow gap without expanding into Lane K:

- inviter sees outgoing community invite evidence in DM on native hydrate;
- invite accept response updates inviter DM status independently of community join publish;
- membership/discovery reads start converging on one explicit read-model owner.

This slice prioritizes honest UI outcomes and owner boundaries over new feature breadth.
