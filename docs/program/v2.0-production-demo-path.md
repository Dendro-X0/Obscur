# v2.0.0 production demo path

**Status:** Active program framing (2026-06-01)  
**Audience:** Maintainer + agents — what “production demo of v2.0.0” means and how v1.9.x tags lead there  
**North star:** [obscur-2.0-milestone-roadmap.md](./obscur-2.0-milestone-roadmap.md)

---

## Goal

Deliver a **production-grade public demonstration** of Obscur v2.0.0 for technical community and B2B trust conversations: working installs, honest docs, demo scripts, and visual evidence — not a marketing push before the product is demonstrably ready.

**Public promotion policy:** Deferred until **after v2.0.0** ([current-session handoff](../handoffs/current-session.md)).
