## Goal

Ship **v1.5.4** as a **desktop/web polish** release with **one Obscur mark** on every **ship surface** — especially Android (and verify macOS/Linux match Windows).

**Bar for success:**

- **Windows** already ships the Obscur mark (shortcuts, NSIS installer, in-app) — treat as baseline, do not regress.
- **Android** launcher / APK icon shows Obscur arcs (not Tauri default).
- **macOS / Linux** artifacts visually match Windows on next tag build.
- At least one additional client stability or relay discipline improvement with regression test.
- **Mobile device testing is suspended** for this version — no M1–M3 gate; APK may still build in CI but is not a ship blocker for maintainer sign-off.
