## 2. Manual REL-004 matrix (desktop A/B)

**Matrix:** [docs/assets/demo/v1.8.3/README.md](../assets/demo/v1.8.3/README.md)

**Setup:** Use a test community on a relay you can block (disable relay in Settings, or disconnect network). Leave durability must hold **without** waiting for relay success.

| ☐ | ID | Step | Pass? |
|---|-----|------|-------|
| ☐ | T4-1 | Leave from group home → confirm | Community gone from Network/groups immediately |
| ☐ | T4-2 | Block relay / offline → leave another community | Still left locally; outbox pending; no re-join |
| ☐ | T4-3 | Restore relay / wait for retry | Pending clears or honest terminal copy |
| ☐ | T4-4 | Settings bulk-leave (if used) | Same as single leave |
| ☐ | T4-5 | Reload app after T4-2 | Left communities stay gone; retry on activation |
