## Tag workflow

```bash
git checkout main && git pull
pnpm version:check && pnpm docs:check
pnpm release:test-pack -- --skip-preflight
pnpm -C apps/pwa typecheck

git tag -a v1.8.11 -m "v1.8.11 — B1 outbound community bot"
git push origin v1.8.11
```

Wait for **Obscur Full Release** → `publish-release`.

**Note:** If desktop assets show an older semver in filenames than the tag, rebuild with a fresh Full Release run (bundle cache + filename normalization fixes in `release.yml`). Re-upload or replace assets on an existing release only after verifying in-app **About** shows the tag version.
