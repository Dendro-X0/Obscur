## Tag workflow (once)

```bash
pnpm version:check && pnpm docs:check
# Optional: workflow_dispatch Obscur Full Release on main first (no tag burn)

git tag -a v1.8.13 -m "v1.8.13 — release CI simplification + R12 parity"
git push origin v1.8.13
```

**Re-run builds without a new tag:** GitHub Actions → Obscur Full Release → Run workflow (leave `release_tag` empty on `main`; uses `package.json` version).

**Publish from main after green verify:** Run workflow with `publish_release=true` and `release_tag=v1.8.13`.
