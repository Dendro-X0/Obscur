## Highlights

### D1 — Relay filter suppress (operator relay)

- Kind **5** hide events index hidden message ids; live `EVENT` frames for those ids are dropped on operator workspace relays.
- **PWA:** `enhanced-relay-pool` + `community-relay-hide-suppress`.
- **Gateway:** `apps/relay-gateway` (optional `pnpm dev:relay:gateway:docker` → `ws://localhost:7000` with D1).

### D3 — Strict workspace mode UX

- Group chats on operator/local workspace relay: **Remove from this workspace** (message menu + batch).
- Honest scope helper; no “deleted everywhere” claim.

### Managed-workspace baseline (from void v1.8.8)

- Test 8: create → invite → accept → sealed group chat.
- Group header, message hydration, relay/leave hardening — see [v1.8.8 release notes](./v1.8.8-release.md).
