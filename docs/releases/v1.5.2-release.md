# v1.5.2 Release Notes

**Codename:** Feel Fast  
**Status:** Released (tag `v1.5.2`)  
**Builds on:** [v1.5.1](./v1.5.1-release.md) (Honest Hide)  
**Scope:** [v1.5.2-scope.md](../program/v1.5.2-scope.md)

---

## Summary (planned)

v1.5.2 improves **perceived performance** and **relay discipline** without expanding product scope:

- Faster first navigation and Network tab switches.
- Community destructive actions on dedicated confirmation pages (no broken modals).
- Relay standby probing and pool connection churn bounded so UI stays responsive.

---

## Planned highlights

### Navigation & shell

- Sidebar route prefetch and client-chunk warmup (including community home when Network loads).
- Shorter page-transition overlays; respect reduced motion.

### Communities

- `/groups/leave`, `/groups/block`, `/groups/purge` confirmation flows.
- Snappier Network dashboard tabs.

### Relays

- Deferred/capped standby latency probes.
- Less duplicate connection work; quieter dev diagnostics.

---

## Known limitations (expected at ship)

- First visit after install may still load large chunks once; goal is **repeat** navigation speed.
- Relay latency on slow networks remains environment-dependent.
- Streaming updater policy may still use fallback until native/same-origin fetch exists.
