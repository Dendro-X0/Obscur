## Tag workflow

```bash
git checkout main && git pull
pnpm version:check && pnpm docs:check
pnpm release:test-pack -- --skip-preflight
pnpm -C apps/pwa typecheck

git tag -a v1.8.12 -m "v1.8.12 — release parity + Android P1 wrap-up"
git push origin v1.8.12
```

Trigger **Obscur Full Release** with `publish_release` after verify-artifacts passes.
