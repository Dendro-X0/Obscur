# v1.8.11 Release Notes

**Codename:** Lane T — B1 outbound community bot  
**Status:** Ready to tag

## Summary

**v1.8.11** introduces **B1**: a steward-controlled **outbound-only bot** for managed workspaces — announcements published through the same sealed community relay plane humans use, visible in group chat without a separate proprietary API. Includes **desktop online reliability** improvements (DM nav cache, relay failover, coordination poll throttle).

**Manual Test B1** (live bot announcement on two clients) is **deferred** to operator environments; automated contract and runner tests pass.

- Sidecar: `pnpm community-outbound-bot` — [community-outbound-bot.md](../messaging/community-outbound-bot.md).
