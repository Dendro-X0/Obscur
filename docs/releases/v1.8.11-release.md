# v1.8.11 Release Notes

**Codename:** Lane T — B1 outbound community bot  
**Status:** Shipped (2026-05-29)

## Summary

**v1.8.11** introduces **B1**: steward-controlled **outbound-only bot** infrastructure for managed workspaces, plus **desktop online reliability** improvements (DM nav cache, relay failover, coordination throttle).

**Known issue:** The GitHub Release desktop installers (exe/dmg/AppImage) were built with **1.8.10** embedded filenames due to stale CI bundle cache. **Android** APK/AAB were correct at **1.8.11**. Use **[v1.8.12](./v1.8.12-release.md)** desktop assets when published, or rebuild from `main` after release workflow fixes.
