## What “v1.9.0” means here

This is the **v1.9.x program band** (Lane K), not a mandate to tag or run the hour-long release workflow. Version `package.json` may stay on **1.8.14** until a **user-visible** milestone bundle is ready.

**Tangible outcomes** (what users should feel):

| Area | User-visible effect |
|------|---------------------|
| **Membership** | Leave on A converges on B when coordination is enabled; honest copy when it cannot |
| **Communities** | Roster no longer collapses to one member after refresh (MEM-001 / R2) |
| **DMs** | Delete-for-me survives reload (DM-001 / R1) |
| **Bots** | Inbound triggers + outbound announcements on managed workspaces |
| **Mobile / UX** | P13/P14 polish — invites, sidebar previews, light theme, tablet layout |

Structural slices (B0–B3, partial B4) support the above; they are not shipped as “empty” version bumps.
