## Summary

**v1.9.0** is the **Lane K band opener** and the first semver in the **v1.9.x** train toward **v2.0.0**. It moves community membership truth into kernel ports and an optional coordination directory, demotes Nostr to transport, and ships honest UI for sovereign vs coordinated modes.

Users should feel:

- **Leave on A converges on B** when coordination is enabled (directory deltas, not gossip hope).
- **Sovereign + public relay** modes show honest limits ‚Äî no false ‚Äúrelay-confirmed member‚Äù claims.
- **Community roster** stays stable after reload (R2 read model; MEM-001 collapse fixed at display layer).
- **Delete-for-me** routes through the DM materialization port (R1; see accepted limitations below).

Structural slices B0‚ÄìB5 are bundled in this tag because engineering exited Phase A‚ÄìC as one concentration unit on `main`.
