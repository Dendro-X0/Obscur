## v1.8.11 caveat (prior release)

The **v1.8.11** GitHub Release shipped **Android** assets at `1.8.11` but **desktop** installers were still labeled **`1.8.10`** due to stale Tauri bundle cache in CI. Use **v1.8.12** desktop installers when available, or rebuild from tag after workflow fixes on `main`.
