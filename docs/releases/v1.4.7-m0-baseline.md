### Reproduction Steps Needed

**To document in `v1.4.7-restore-repro-steps.md`**:

1. Create account with media in conversations
2. Backup to relay/coordination
3. Fresh browser/device (clear IndexedDB)
4. Restore account

6. Check: Are media blobs in Vault?
