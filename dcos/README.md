# DCOS

This directory contains developer-facing documentation for Obscur.

## What is Obscur?

Obscur is an invite-only, local-first social/messaging app built on Nostr transport primitives. The product goal is to make small, trusted communities feel safe and usable without accounts, passwords, or centralized profiles.
